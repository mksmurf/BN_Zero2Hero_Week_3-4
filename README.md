# 代码说明

**逻辑合约 Smurf.sol**

v1版本部署
```shell
npx hardhat run scripts/1.deploy_smurf.ts --network localhost
```
与v1版本交互
```shell
npx hardhat smurfV1 --network localhost
```
升级v2
```shell
npx hardhat run scripts/2.deploy_smurfV2.ts --network localhost
```
与v2版本交互
```shell
npx hardhat smurfV2 --network localhost
```
